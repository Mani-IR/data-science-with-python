## 📖 Description | توضیحات

**English:**
This repository contains a collection of hands-on exercises for the **Data Mining** course. Each exercise focuses on a specific topic in data mining and machine learning, implemented in Python using popular libraries such as pandas, scikit-learn, matplotlib, and seaborn. The exercises are based on the `airport-codes.csv` dataset.

**فارسی:**
این ریپازیتوری مجموعه‌ای از تمرین‌های عملی درس **داده‌کاوی** را شامل می‌شود. هر تمرین روی یک موضوع خاص در داده‌کاوی و یادگیری ماشین تمرکز دارد که با پایتون و کتابخانه‌های محبوبی مثل pandas، scikit-learn، matplotlib و seaborn پیاده‌سازی شده‌اند. تمرین‌ها بر اساس دیتاست `airport-codes.csv` هستند.

## 📊 Exercises Overview | نمای کلی تمرین‌ها

### Exercise 1 | تمرین اول
**Statistical Analysis & Data Visualization | تحلیل آماری و مصورسازی داده**
- Descriptive statistics (mean, median, mode, variance, skewness, kurtosis)
- آمار توصیفی (میانگین، میانه، مد، واریانس، چولگی، کشیدگی)
- 10+ types of plots (histogram, boxplot, scatter, heatmap, pair plot, etc.)
- بیش از ۱۰ نوع نمودار (هیستوگرام، باکس پلات، پراکندگی، هیت‌مپ، جفتی و ...)

### Exercise 2 | تمرین دوم
**Data Preprocessing | پیش‌پردازش داده**
- Data cleaning (missing values, duplicates, outliers)
- پاکسازی داده (مقادیر گمشده، تکراری‌ها، پرت‌ها)
- Transformation (normalization, standardization, encoding)
- تبدیل (نرمال‌سازی، استانداردسازی، کدگذاری)

### Exercise 3 | تمرین سوم
**Clustering Comparison | مقایسه خوشه‌بندی**
- K-Means Clustering
- خوشه‌بندی K-Means
- DBSCAN Clustering
- خوشه‌بندی DBSCAN
- Silhouette Score evaluation
- ارزیابی با Silhouette Score

### Exercise 4 | تمرین چهارم
**Classification | طبقه‌بندی**
- Decision Tree Classifier
- طبقه‌بند درخت تصمیم
- Naive Bayes Classifier
- طبقه‌بند Naive Bayes
- Accuracy comparison
- مقایسه دقت
